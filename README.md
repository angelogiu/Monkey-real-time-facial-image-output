## About The Project

This project is a Python application that uses OpenCV and MediaPipe to track face and hand movements in real time through your webcam. Based on your expressions and gestures, the program dynamically displays a corresponding monkey image that mimics your movement. It’s a fun and interactive demo combining computer vision, live tracking, and playful visual output.



<!-- GETTING STARTED -->

### Prerequisites

These are some of the libraries you need to install before running the program:

  ```sh
   pip install numpy
   pip install opencv-python
   pip install mediapipe
  ```
